DOM -> Document Object Module

DOM

  • It contains all the methods and properties of a website and use to interact with a website.
  • Representation of a website
  • It containlot of method to interact with website
  • All content and methods are used in DOM

  • getElementbyTagName

    1. data of that tag
    2. document.getElementByTagName('target')-syntax to search in console.
    3. The output will be in a form of an array -> like HTML collection

  • getElementByClassName

    1. data of that class
    2. document.getElementByClassName('class name')-syntax to search in console.
    3. The output will be in a form of an array -> like HTML collection

  • getElementById

    1. data of that Id
    2. document.getElementById('Id name')-syntax to search in console.
    3. The output will be in a form of an array -> like HTML collection

  • querySelector

    1. We can use parent child relation
    2. document.querySelector('')-syntax to search in console.
    3. output will be unique
    4. Output will be first occured element
    5. similar to css

  • querySelectorAll

    1. We can use parent child relation
    2. document.querySelectorAll('')-syntax to search in console.
    3. output will be unique
    4. Output will be all element
    5. similar to css

  • Class List
    • It is only used for class.

    • Types of class list

      • Add - Used to add the data in a class.
      • Remove - used to remove the data in a class.
      • Contains - used to sheck the data is in that class.
      • Toggle - used to add the data if it is not in the class and also used to remove the add if it is in the class.
